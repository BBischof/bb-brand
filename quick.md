# Bryan Bischof Brand — Quick Reference

| Role        | Light     | Dark      | Meaning                          |
|-------------|-----------|-----------|----------------------------------|
| Background  | `#F5F4F1` | `#141412` | default surface                  |
| Surface     | `#ECE9E4` | `#1C1C19` | cards, panels                    |

| Text muted  | `#6D6A64` | `#A7A093` | labels, metadata                 |
| Accent      | `#D97A32` | `#E08A43` | featured / curated / authored    |
| Interactive | `#5F8A3B` | `#91B85A` | live UI / explicit active states |
| Analytic    | `#7A4FB3` | `#A47DDB` | data / math / comparison         |
| Critical    | `#B6423C` | `#E06A63` | errors / warnings                |
